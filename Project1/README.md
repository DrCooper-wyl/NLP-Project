# Project #1

## How to use
Linux:
```shell
pip3 install -r requirements.txt
python3 spider-url.py
python3 spider-books.py
python3 preprocess.py
```
or
Windows:
```shell
pip install -r requirements.txt
python spider-url.py
python spider-books.py
python preprocess.py
```
